# 04 · 成本与运维铁律（通用，最高优先级）

> 这些是**花钱 / 不可逆 / 安全**相关的红线，跨所有模板生效。宁可停下问，也不许自作主张烧钱或提交大文件。

## 一、★★★ fal.ai 出问题不许擅自重调（用户锁定，最高优先级）

- **每一次 fal.ai 调用都花钱。** 若某次 fal 调用（出图/编辑/任何端点）结果有问题（黑图/画错/画风偏/不满意），**禁止 Claude 自己"再试一次"**。
- **必须先把问题告诉用户、等用户明确同意，才能再调 fal。** 用户没点头就停在那，不许自作主张烧钱重试。改 prompt/参考图后想重试，也要先经用户同意。
- **免费/本地步骤不受限**、可自由重跑：TTS（火山）、ASR、本地合成 BGM/SFX、Remotion 渲染、ffmpeg 剪辑、改 config/脚本。

## 二、★ 音乐一律本地合成，禁止任何 API/fal 生成

- BGM 只能 **①下载开源真实录音**（CC0/公有领域，音色地道）或 **②本地纯代码合成**（`scripts/gen-bgm.py`，numpy 合成、固定种子可复现）。
- **绝不调用 fal / 任何 API 生成音乐。** ⚠️ Claude 听不到声音，纯合成盲调难到位——优先真实录音。
- 接线：`template.json → audio.bgm = { src, volume }`；渲染层整片低音量循环垫在旁白/原声之下。换音乐只改 config 或替换文件，不动代码。

## 三、★ mp4 一律 gitignore，禁止提交上传

- 所有 `*.mp4`（含成片、剪辑源、part 片段）一律在 `.gitignore` 中禁止提交。成片只在**本地生成、预览、交付**。
- 其它产物（图/音/manifest/script）可入 git 便于分发（代价：仓库变大）。

## 四、出图模型路由通则（仅"出图类"模板适用）

严格**依据文案/分镜里出现的人物数**定"画面几个人"，**不得臆造增删**，再按人数路由：

| 画面人物数 | 模型 | fal id | ~$/张 | 参考图 |
|---|---|---|---|---|
| **0 人（空镜/物）** | FLUX.1 Kontext [pro] | `fal-ai/flux-pro/kontext` | ~0.04 | 喂**风格锚图**只借画风、绝不加人（**非纯文生图**，无锚会漂移/乱加人） |
| **1 人（单角色）** | FLUX.1 Kontext [pro] | `fal-ai/flux-pro/kontext` | ~0.04 | 喂该角色 1 张定妆图 |
| **≥2 人（多角色同框）** | Nano Banana Pro edit | `fal-ai/nano-banana-pro/edit` | ~0.15 | 喂多张定妆图 |

- 🚫 **严禁 `nano-banana` 文生图、`flux/dev`**（无锚漂移/乱加人）；空镜也走 kontext 喂风格锚图。`gen-image.mjs` 强制校验 `model` 与人数一致，不符报错。
- 🚫🚫 **一条视频最多 1 次 banana**：全片至多一个多角色拍（最多 1 次 `nano-pro` $0.15），其余全 flux($0.04)；build 出图前统计 >1 直接报错。想多处出现多人 → 改单人特写/空镜表达。
- 🚫 **死命令**：prompt 里**绝不**写 box/frame/subtitle/"下方留白给字幕"/"empty area at bottom"（含 `no box` 负面写法）——扩散模型"提到什么画什么"会真画出框；也**不放身体部位负面词**（`no extra fingers` 等会触发 fal nsfw 黑图）。留白靠"主体 + 大片纯白 + 四周留边"自然得到。
- 出图**一律纯白底 `#ffffff`**；出图后用 sharp 量人物身高做**尺寸归一化**（所有拍人物一样大）——白底是量轮廓的前提。
- **成本预算**：≤ $1 / 分钟视频（出图为主，语音成本忽略）。按 hash 缓存 + 场景共图 + 角色/风格锚全局复用，把边际成本压到最低。

## 五、fal / 火山 调用链路

- **一律 curl + 代理 `http://127.0.0.1:7897`**（Node fetch 直连会超时/不走代理，见记忆 [[fal-use-curl-proxy]]）；本机 7897 未监听时脚本自动直连重试。
- 凭据取仓库根 `api-key.txt`（火山 APP ID / Access Token；fal Key 取相应行）。
- ⚠️ node 在 `.sh` 里读中文文件名会 ENOENT → 参考图用 ASCII 名。

## 六、出片前逐句体检（免费、必做）

- 逐句听多音字读音，读错就补引导字段重合成该拍（只烧这一句，见 [[03-流水线与manifest]] 四）。
- 对照留存/完播的目标，走一遍 [[01-创作准则]] 八的通用自检清单。
