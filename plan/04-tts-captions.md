# 04 · 配音、字幕、翻译、拼音

## 配音（火山引擎）
- 音色：**开朗姐姐**，旁白叙述。永远中文，不随字幕语言变。
- 凭据见项目根 `api-key.txt`（豆包/火山 APP ID / Access Token / Secret Key）。
- 每段中文旁白单独合成一个 `.mp3`，取**真实时长(ms)** → 驱动该页停留与字幕时间。
- 段尾加少量留白（呼吸），避免翻页过紧。

## 关键待查：字级时间戳
- 查火山 TTS 是否返回**字/词级时间戳**。
  - 有 → 中文字幕做**逐字高亮（卡拉OK跟读）**，对中文教学价值很高。
  - 无 → 整句淡入淡出，停留 = 整段音频时长。
- 见 [[06-open-questions]]。

## 翻译（当地语言，首发越南语）
- 用 LLM 翻译，明确要求：**儿童口语、自然、贴合家庭日常**，不要直译腔。
- 配置驱动：`lang=vi/th/...`，加语言只加翻译目标。

## 拼音
- 带声调（ā á ǎ à），按词分隔便于阅读。
- **多音字坑**：行/重/长/得/和/觉 等按词义选音。需用可靠拼音库 + 必要时 LLM 校正词义读音。

## 背景音乐 / 音效
- 轻柔儿童向 BGM（注意版权，用可商用/无版权源）。
- 翻页"唰"音效，配合 [[03-remotion-animation]] 翻页转场。
