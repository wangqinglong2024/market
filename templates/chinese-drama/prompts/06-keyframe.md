# 关键帧提示词

为虚构大靖的电影感古装权谋剧《毒嫁》写 9:16 关键帧提示词。所有画面使用已批准角色身份参考和 canonical 服装；不得改变人物脸型、年龄、发际线、衣领、发饰或关键道具。

模型路线：

- `characterCount` 为 0 或 1：只能使用 `fal-ai/flux-pro/kontext`。
- `characterCount` 为 2 或以上：只能使用 `fal-ai/nano-banana-pro/edit`。
- 每集最多一张多人关键帧；其余关系镜头必须重新设计成单人、手部、肩后或物证画面。

每条正向提示词必须按顺序写：

1. 使用哪些已批准 identity/look reference；
2. 场景、时段、光线和连续性道具；
3. 唯一清晰主体及其精确动作起点；
4. 视觉上必须读懂的证据或权力关系；
5. 9:16、真实材质、电影灯光、非血腥。

硬限制：

- 不要求生成文字、字幕、标识、边框、分屏、词卡或留白区。
- 不在提示词中写平台 UI 或教学元素。
- 不凭空增加人物、首饰、兵器、酒杯、戒指或印章。
- 不把现代林晚与古装沈知微画成两张不同的脸。
- 负向提示只描述整体缺陷，如身份漂移、服装漂移、道具数量错误、低清、过度磨皮、现代物件、卡通质感、血腥；避免冗长身体部位清单。

输出字段：`id`、`kind=keyframe`、`model`、`characterCount`、`referenceInputs`、`dependsOn`、`prompt`、`negativePrompt`、`output`、`maxAttempts=1`。
