# chinese-drama 写本铁律（剧本层·连续剧）

> 出图层铁律见 [[render-rules]]；剧情事实源见 [[../story/00-bible]]（爆点版=[[../story/new_story/00-bible]]）。以下为强制约束，违反即废稿。
> **写任何一集前，先读 base/[[../../../base/00-底层规律]] 和 base/[[../../../base/01-创作准则]]（全项目最高事实来源），再读本文。**

## 0. ★黄金3秒·爆点闸门（用户 2026-07-16 锁·第一优先·不过此关不许写正文）

3秒定生死，70%的人在读到台词之前就划走（base 00·二.1、三.2）。**开头是唯一真正重要的部分，先设计开头再写别的**（base 01·一）。每集开场必须逐条过闸：

1. **第0–1.5秒引爆"本集爆点"**：首帧+首句直接架在本集最炸的名场面上（见 arc「💣本集爆点」）。**禁止铺垫/热身/过场开场**——"那夜在大叻夜市""纸和笔摆在面前""医生进来了"这类**零触发器的环境/道具/过程句一律废稿**（base 01·一.3）。
2. **触发器≥2**：第一句命中≥2个触发器（与我有关/钱与利害/好奇缺口/高唤醒/社交货币）。**本剧最强弹药：与我有关（越南女="我"）＋生死利害＋好奇缺口**。<2 个 → 回炉（base 00·一 叠加规律）。
3. **每集自证·禁裸承接**：前3秒必须让**没看过前集的人**也懂"越南女假冒豪门夫人＋此刻的死局"。给每集一句"一句话前情钩"，"上集说到…"式承接开场废稿。
4. **首帧=首句台词的戏剧内容**（画面=台词，见第4条），且**禁弱题材帧**（落水/写真/空镜/摆拍当第一帧废）。要么最炸的画面，要么最勾的字幕（base 01·一.2）。
5. **前传后置**：坠湖/换魂等来龙去脉不许当开场，钩住之后用一闪回交代（base 01·一）。
6. **答案后置**：本集爆点的结果压到本集最后一秒；本集尾钩=下集头钩（base 00·二.3/二.4）。
7. **落地口径**：每集 script.json 的 `_comment` **第一行必写「💣本集爆点：…」**，开场前2–3拍逐字落实引爆句；写完在自检清单第0条复核。

## 1. 女主中文水平（锁定人设）：懂一点，但水平不高

- 灵兰**不是零中文**，是**半吊子初学者**（来中国前在大叻旅游区卖花、看中国剧，耳濡目染过一点）：
  - **能**：听懂高频短词短句（你醒了/医生/明天/谢谢/吃饭/多少钱…），从慢速短句里抓关键词、猜个大概；
  - **不能**：长句、快语速、书面语、成语一概跟不上；开口只能蹦单词，说不成句，声调常错。
- **水平逐集上升**：以各 arc 文末「中文课程表」为准。第 N 集她会的 ≥ 前几集已学会的全部，不许倒退、不许跳级。

## 2. 反应必须与听懂程度一致（逻辑自洽铁律）

- **她对一句话的内容做出反应 ⇔ 她听懂了那部分**，二者必须同真同假：
  - 错例：前面写「我一个字都听不懂」，后面她却知道「医生明天来检查」——她既然全听不懂，怎么知道对方说的是医生？逻辑错乱，废稿。
  - 对例：顾晏辞说「医生明天来检查你」→ 她抓住「医生」「检查」两个词 → 才有资格慌「医生要来检查我？」。
- 她听不懂/半懂时的内心戏写**情境困惑**，不写"全听不懂"：
  - 对例（第1集听到「你醒了」）：「这是哪里？这个人为什么在说中文？」
  - **全剧禁句**：「我一个字都听不懂」「完全听不懂」这类与人设矛盾的表述（她主动装傻说给别人听的「听不懂」除外，那是台词武器不是事实陈述）。

## 3. 设定措辞（穿越描述）

- 穿越 = 灵魂进入别人身体。旁白/台词一律写「**灵魂**在别人的身体里」，禁止「身在别人的身体里」这类含混措辞。
- 涉及世界观的措辞（换魂/玉镯/双魂）以 [[../story/00-bible]] 为准，不得自创说法。

## 4. 画面必须演出台词（画面=台词铁律）

- 每个 sceneId 一张图，这张图必须**演出该场景所有拍的台词动作与情绪**，不是氛围配图：
  - 台词说「坠入湖中」→ 画面必须是坠湖瞬间（人在半空/砸进水面/惊恐挣扎），不能是安静泡在水里微笑；
  - 台词说「这不是我的手」→ 画面必须是她惊愕地抬起双手盯着看。
- 写 shot.content 的固定流程：**先列出该场景全部拍的 captions → 提取关键动作+关键情绪 → 把动作、表情、机位逐一写进 shot**。缺动作或缺情绪的 shot 视为未写完。
- 一张图盖不住同场景多拍的关键动作 → **拆场景**（新 sceneId、多出一张图），不许硬凑。

## 5. 连续性（禁止胡编铁律）

- 写第 N 集前**必读**（顺序）：
  1. `story/00-bible.md`（世界观/人物/总图）；
  2. `story/arc-XX.md` 里第 N 集的梗概与前后集尾钩（本集头钩=上集尾钩）；
  3. **前 1–3 集已产出的 script.json**（public/videos/…/linglan-epXX），核对既成事实。
- 必须延续的既成事实：她已学会的中文词、每个人物各自知道/不知道什么、道具状态、服装（同场景连戏，见 render-rules 2）、场景地点与时间线。
- 与前情冲突 = 废稿。arc 梗概与 bible 冲突时以 bible 为准；发现冲突先报告用户，不许擅自改设定。
