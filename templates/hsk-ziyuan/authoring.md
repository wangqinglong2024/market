# 模板：hsk-ziyuan · HSK字源·古风美人识字（越南受众 · 9:16竖屏）

> **输入物 = 一组单字 + 每字的意境场景**（区别于漫剧的「文案」）。产出一条「看画猜字 → 水墨写成字」的单字合集短视频。
> 做本模板前先读 base 的 [[00-底层规律]] 与 [[01-创作准则]]，本文是它们在「HSK 单字」上的落地。

---

## 一、这条片子赢在哪

不是「教一个字」，是设计一条**越南人拇指停下、划不走、看到最后还记住几个字**的时间线。

- **北极星**：3 秒留存 → 完播 → 评论。触发器叠加：**与我有关**（我在学中文）+ **好奇缺口**（这是什么字？猜）+ **高唤醒**（美到窒息的古风画面 + 水墨写字的「啊哈」）+ **社交货币**（6 张能当壁纸的画面、「你认识几个」）。
- **格式是资产**：这是可复利的系列格式。每条是「同格式的下一集」，不做孤品。
- **护城河**：AI 让「一幅画当场水墨写成一个汉字」——图库做不出的变形，是我们的不公平优势。

---

## 二、唯一形态（★方案 B · 定死，别自由发挥）

**古风男/女主（锁脸）立于山水意境中（人在画里）→ 身后景物水墨晕开、逐笔写成对应汉字（字从景里浮现，不剧透）→ 人字同框、发音落在写成那刻。**

- **男女主轮流，男主先**：第 1 字男主、第 2 字女主、交替。发音跟人走（男=清爽男大 / 女=温柔小雅）。
- **每字只念这一个字**：`tts` = 单字（如「山」），**不组词、不加例句**（组词塞不进 2 秒）。
- **一字一拍 ≈ 2 秒**，6 字 ≈ 12–13 秒（短完播友好，见 [[00-底层规律]] 二.2）。
- **一条 = 一个主题合集**（如自然组 山水火木日月），成套才有合集钩子；随机字 = 陈述结构会被划走。

### 一拍三段（渲染层 src/layouts/hsk-ziyuan.tsx 免费叠）

| 段 | 时长(默认) | 内容 |
|---|---|---|
| ① 猜 | `guessMs` 600ms | 实景空镜·人在画里，底部越南语 `guessText`「这是什么字?」。字未现，不剧透。 |
| ② 写 | `writeMs` 700ms | 水墨晕染渐显；大毛笔字从上到下逐笔显形（clip-path）+ 由虚渐实（blur）。观众跟着猜。 |
| ③ 定 | 至拍尾 | 人字同框；拼音（上）+ 越南语释义（下）弹入；**发音在此刻响**（延到写成那刻）。 |

时长：`durationMs = max(charTargetMs, guessMs+writeMs+发音时长+tail)`，保证发音放得完。

## 三、版面（固定）

- 画布 **1080×1920（9:16）**，宣纸米黄基调，沉浸满帧场景图 + Ken Burns 运镜。
- 汉字簇居**上部负空间**（拼音 / 大毛笔字 / 越南语释义）——所以**出图 prompt 必须把人放在下三分之一或侧边，上中部留白给字**（见 prompts/style.md）。
- 水墨晕染（radial 深色）在写字段渐显，给毛笔字一块深底，保任何画面都读得清；人物在晕染外仍可见 = 人字同框。
- 字体：汉字 Ma Shan Zheng（毛笔行楷）；拼音/越南语 Charm。换字体只改 template.json `fonts`。

## 四、生产流程（★闸门 + 花钱铁律）

1. **定妆图闸门（先审脸）**：`node templates/hsk-ziyuan/portrait.mjs gongzi` 和 `... meinv` 各出一张锁脸定妆图（nano-banana-pro 文生图，~$0.15/张）。**人工审脸满意后才继续**。不满意→删图、改 characters/<id>/canonical.md、**经用户批准**后重跑。
2. **出场景 + manifest**：`node scripts/build.mjs hsk-nature-01`。每字喂该拍主持人定妆图走 nano-edit 出 9:16 场景图（~$0.15/张 × 6 = $0.90），配音、落 manifest。
3. **渲染**：`npx remotion render hsk-nature-01 <out>.mp4`（或 `node scripts/run.mjs hsk-nature-01 --render`）。

- **花钱铁律**（[[fal-recall-needs-approval]]）：fal 出问题（黑图/画错/不满意）**不自动重试**，报出来等用户批准再重调。每张只允许一次尝试。
- **预算硬卡**：`budget.maxUsdPerEpisode`，build 累计超上限立即抛错停。
- 全程 hash 缓存：改文案/换字/调时长免费重渲；图片缓存不重复计费。

## 五、写一条新的（下一集）

1. `catalog.json` 加条目（`template: "hsk-ziyuan"`，新 `id`/`shard`）。
2. `public/videos/<shard>/<id>/script.json`：定 `hostRotation`、`guessText`、6 个 `beats`（char/pinyin/gloss/host/tts/scene/motion）。
3. **选字规矩**：只做「能画出来」的字（象形/会意/指事）——图变字的魔法只对这类字成立。成主题合集。
4. `scene` 写法：设定 + 换装配色 + 人物置于下三分之一/侧边（上中留白）；性别与该拍 host 一致（男=公子长袍，女=高开衩长裙露腿，国风高级不擦边）。
