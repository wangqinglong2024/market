# 模板：hsk-ziyuan · HSK字源·九宫格识字（越南受众 · 9:16竖屏）· 定稿

> **输入 = 分组的「汉字 + 越南语释义」**；骨架/拼音/发音自动从字形库与发音库取。产出「具象简笔画 → 线条字 → 真实毛笔字」的九宫格单字合集。
> 做本模板前先读 base 的 [[00-底层规律]] 与 [[01-创作准则]]。本文是它们在「HSK 单字」上的落地与定稿规则。
> **★子模板清单见 `SUBTEMPLATES.md`**（纯字版 plain / 炸裂墨韵版 inkburst），生产视频时由用户指定子模板名。全片仅单字中文朗读，无越南语配音。

---

## 一、为什么这是护城河

竞品做单字靠 CapCut 手拼静图。本模板是**代码驱动的字源 morph 引擎**：字形写一次进 `glyphs.json`，之后所有视频免费复用；每个字都有量身定制的「具象→字」几何演变——手工做不到这个规模。**门槛在引擎，不在某条视频。纯代码渲染，单条制作成本≈0（仅 TTS）。**

## 二、唯一形态（★定死）

**9:16 竖屏 · 内容区 2×2 共 4 格（一字一格）· 每组 4 字在整组时间内「同时、逐点、连续」演变。**

一个字的三段演变（每组 4 字**完全同步**进行）：
1. **具象简笔画**浮现（draw-in）——★**必须是实物的样子**（`日`=一轮太阳、`山`=山峰、`木`=树、`目`=眼睛）。
2. **前段**：简笔画用**同一套线条工具**逐点几何演变成「线条字」（顺其自然，无突变，不用毛笔字切换）。
3. **末段（约末 1 秒）**：线条字 → **真实毛笔字**（Ma Shan Zheng）。此段同时**修正**了难字线条形不准的问题。

**朗读独立于演变**：1 字 1 拍、**每字 1.2 秒**顺序朗读（爽快思思）；读到哪个字，哪个字才触发**水墨韵律**（墨晕环扩散 + 脉动 + 辉光）。第 1 字在其被读时可能尚未成形——**朗读与演变是两层，互不绑定**。

**默认 2 组 = 8 字 = 9.6 秒**（短→完播高，[[00-底层规律]] 二.2）。一条 = 一个主题合集。

## 三、★铁律：象形起点必须具象

`pic`（起始简笔画）画的是**实物**，不是字。反例（❌）：`日` 一开始就画成方框（≈字本身），毫无"由画变字"的揭示。正例（✅）：`日` 起点是**圆太阳+光点**，再演变成方形「日」。加任何新字，先问："它一开始画的是那个东西吗？"

## 四、版面（固定，1 格 = 120px）

- 画布 1080×1920。上下留白各 2 格（内容 y0=240、高 1440）；左右各 0.5 格（x0=60、宽 960）。
- 2×2 共 4 格，每格 480×720。田字格边长 340 居中（带米字辅助线），汉字/简笔图**同框、垂直居中**格内。
- 每格：**拼音在上（青碧）· 字在中 · 越南文在下（深红）**。字体：汉字 Ma Shan Zheng，拼音/越南文 Nunito（柔和、完整越南语声调）。

## 五、数据与流水线

- **字形库** `glyphs.json`：`{ 汉字: { py, pic, chr } }`。`pic`/`chr` 必须**同构**（相同笔数、每笔相同点数）才能逐点插值。坐标系 0..100（y 向下）。**可复用、跨视频。**
- **发音库** `public/library/tts-hanzi/<han-hex>.mp3`：单字朗读，hash 缓存跨视频复用。
- **视频 script.json**：`{ groups: [ [ {c, vi} ×4 ], ... ] }`。只写汉字 + 越南语释义；其余自动取。
- **构建**：`node scripts/build.mjs <id>` → 查库 + 配音 + 出 manifest；`npx remotion render <id> <out>` 渲染。参数（配色/时序/音色/字号）全在 `template.json`，改它即改行为，不动脚本。

## 六、封面

`cover-hsk` 组件：宣纸 + 越南语钩子 + 副标题行 + 三个「具象→字」示例 + 系列标，各子模板共用、props 驱动（hook/sub/chars/ep/tag）。每条视频目录放 `cover.json`，各子模板的封面文案口径与渲染命令见 **`SUBTEMPLATES.md` 封面规则**。渲染：`npx remotion still src/index.ts cover-hsk <视频目录>/cover.png --props=<视频目录>/cover.json`。

## 七、加一条新视频（下一集）

1. 用户指定子模板（`SUBTEMPLATES.md`），需要的字若不在 `glyphs.json` → 先补其 `pic`/`chr`（同构、pic 具象）。
2. `public/videos/<shard>/<id>/script.json`：按子模板示例写分组的字 + 越南语释义（+引导条等开关）。
3. `catalog.json` 加条目（`template:"hsk-ziyuan"`）。
4. build + render 成片 + `cover.json` + render 封面（四件套齐才算完）。**选字规矩**：只做象形/指事/会意等「画得出实物」的字；成主题合集。
