# hsk-ziyuan 子模板清单

> 生产视频时**用户指定子模板名**，按对应示例写该视频的 `script.json`。
> 所有子模板共用同一套构建管线（`build.mjs`）与渲染层（`src/layouts/hsk-ziyuan.tsx`），差异只在 script.json 的配置组合。
> 全片仅单字中文朗读（爽快思思），**无任何越南语配音**（2026-07-20 用户定稿）。

| 子模板名 | 中文名 | 模式 | 节奏 | 典型时长 |
|---|---|---|---|---|
| `plain` | 纯字版 | parallel | 4字**同时·线性匀速·严格同步**演变（全程4秒一起动）；朗读与动画**分离**=1秒一个，读到哪字哪格特殊显示 | 4s/组，3组=12s |
| `inkburst` | 炸裂墨韵版 | inkburst | 冷开场1.5秒直接炸出「日」再读 + 3组×3.5秒（前2秒线性匀速变化→后1.5秒四格一齐炸成「墨炸弹」）；朗读每字≤0.75秒 | 冷开场1.5s+3组×3.5s=12s |

## plain · 纯字版

只有字，没有任何引导/提示文字。**动画与朗读分离**：4 字**同时·线性匀速·严格同步**「简笔画→线条字→毛笔字」演变（全程 4 秒一起动，逐帧进度完全一致；morph 用线性不用 ease，避免「有的快有的慢」；被读的字不加脉动缩放）；朗读=**1 秒一个**，读到哪字哪格「特殊显示」（朱晕+朱光+拼音/越南文跳出）。每组 4 秒（`groupFrames=120`、`readFrames=30`），3 组=12 秒。

```json
{
  "groups": [
    [ { "c": "山", "vi": "núi" }, { "c": "木", "vi": "cây" }, { "c": "日", "vi": "mặt trời" }, { "c": "月", "vi": "trăng" } ],
    [ { "c": "水", "vi": "nước" }, { "c": "火", "vi": "lửa" }, { "c": "雨", "vi": "mưa" }, { "c": "目", "vi": "mắt" } ]
  ]
}
```

示例视频：`hsk-ziran-01`。

**封面**（cover.json，规则见文末「封面规则」）：钩子 = 字源揭示（如 `Chữ Hán bắt nguồn từ hình vẽ`），`ep` = `Tập N`（第 N 集）。

## inkburst · 炸裂墨韵版

在 `plain`「4字同时演变」基础上升级为**墨炸弹**：morph 用 easeIn **慢起急收**制造蓄势，第 `burst` 帧（=morph 终点）**四格一齐引爆**。继承 app 水墨语义（`template/game/01-ink-wash`），用 Remotion/SVG 把「象征」升成「真墨」——渲染层 `CellInkburst`（`src/layouts/hsk-ziyuan.tsx`）：

- **★无任何圆环 · 不规则随机墨迹炸裂开**（旧版「同心圆扩散」+「冲击闪圆环」已被否删）：
- **中央巨墨爆体（BANG）**：frame0 一记浓墨硬核砸碎，`feTurbulence`+高位移打散成不规则块（非正圆），短促（<5帧）交给墨弹撑场；
- **墨弹飞溅（主体）**：数十枚墨弹 frame0 **一齐**、**全随机角度**高速甩飞——三档 `chunk`(巨墨块·主爆体) / `shard`(中·多带飞白甩条) / `mist`(细沫)；初速极高→空气阻力急停（`reach = v0·τ·(1-e^-t/τ)`），飞行段速度大时被拉成**彗尾/飞白甩条**，减速后收圆；过半墨弹落点收成**不规则墨渍留在宣纸上**（aftermath），其余飞散淡出；朱砂点缀少量；
- **毛笔字被砸进画面**：`scale` 弹簧过冲 punch（1.3→1.0）+ 飞白由糊变锐（blur 递减）；
- **整格硬震**：爆瞬最猛、~8 帧指数衰减。

节奏（**12 秒** = 冷开场 1.5s + 3 组×3.5s，比上版单字/每组各 +0.5s，全加在爆炸停留上）：每组 `groupFrames=105`、前 2 秒**线性匀速·严格同步**变化（`draw[3,22]`+`morph[22,60]`）→ 后 1.5 秒炸裂（`burst=60`、`EXP=45`）；朗读 `readFrames=22`（每字≤0.75s，`reads 2/24/46/68`），读到哪字哪格「特殊显示」。**`script.intro` 冷开场字统一用「日」，且不得与该集 groups 重复**（`frames=45`=1.5s，直接炸出字再读）。参数在 `template.json → grid.inkburst`。

```json
{ "mode": "inkburst", "intro": "日", "groups": [ "…3组×4字（不含「日」）…" ] }
```

示例视频：`hsk-burst-05`（冷开场日+12字3组=12s，shard 2026/07/21）。

**封面**（cover.json）：钩子 = 字源揭示（与 plain 同口径，汉字由图画演变而来），`ep` = `Tập N`。

## 封面规则（各子模板共用）

组件 `cover-hsk`（`src/CoverHsk.tsx`）：宣纸底 + 越南语钩子 + 副标题行 + 三个「具象简笔 → 毛笔字」示例 + 底部系列标。各子模板共用，差异全在 props。
**★版面居中铁律（2026-07-20 用户定）**：所有内容压进画面中段（y≈440..1440），上下各留 ≥440px 空白安全区——TikTok 顶部搜索栏/底部文案会遮挡上下边缘，封面文字不许贴边。

1. 每条视频目录放 `cover.json`，字段：
   - `hook`：主钩子（墨色 84px，1 行为佳）——子模板各有口径，见上方各节；
   - `sub`：副标题行（青碧 46px）——plain 缺省为 `山 = ⛰️ · 日 = ☀️ · 木 = 🌳`；
   - `chars`：三个示例字（缺省 `山/日/木`，可换成该集的字，须在 `glyphs.json` 中）；
   - `ep`：系列标（深红）；`tag`：固定 `HSK · Tự học tiếng Trung`。
2. 渲染：`npx remotion still src/index.ts cover-hsk <视频目录>/cover.png --props=<视频目录>/cover.json`
3. 生产一条视频 = script.json + cover.json + 渲染成片 + 渲染封面，四件套齐才算完。

## 新增子模板的规矩

1. 先在本清单登记：名字、模式、节奏、示例 script。
2. 参数（几何/配色/字号/音色）一律进 `template.json`，script.json 只放该视频的内容与组合开关。
3. 象形起点必须具象（见 `authoring.md` 铁律），新字先补 `glyphs.json`。
