## 一、一句话卖点
**现代顶级女法医，一朝穿越成罪臣之女，被送进摄政王府"冲喜"——她要用现代法医的眼睛，
在深宅与朝堂之间查冤、复仇、先婚后爱，步步为营谋一个活路。**

- 受众：**越南**（后续可复制泰国等东南亚）。
- 呈现：竖屏 3:4，**30 秒/集**（成片实测基准，见 `public/videos/.../huangmou-e001-kling`），**图文 + 动态视频（kling I2V）随机穿插**，前 3 秒必是动态钩子。
- 语言：**旁白越南文**（受众好跟剧情）+ **角色对白/女主内心中文**；字幕三行 拼音/中文/越南文 逐字卡拉OK。
- 画质：**全部出图（定妆图 + 剧集关键帧）用 nano-banana-pro**（真人剧照质感，已弃用 flux）。
- 成本：**≤ $3/集**（nano-banana-pro 关键帧 + kling 动态 + 海报封面）。

## 二、世界观
架空王朝 **大靖**，承平日久而朝纲暗蚀，摄政王与丞相两强相争，少帝年少多疑。
民间有"冲喜"陋习：贵人病危纳女成婚"冲煞"，多为陪葬弃子。
- **靖王府**：摄政靖王裴珩的府邸，因王爷"病重"而门庭冷清，实为权力风暴眼。
- **苏家**：清流官宦，三月前以"通敌叛国"罪满门下狱抄家；嫡女苏昭华被以"冲喜"名送进靖王府，实为人质弃子。
- **朝堂**：丞相卫崇把持中枢、构陷清流；裴珩装病蛰伏、暗中布局；少帝居中猜忌。
- 质感：真人电影级古装剧（非动漫/非特效堆砌），宫廷+深宅+市井三层空间；仙侠元素只在江湖支线轻量出现。

## 三、主角（双身份，一张脸的现代魂）
**id `zhaohua`｜现代：陆黎 Lù Lí｜古代：苏昭华 Sū Zhāohuá（唤"昭昭"）**
- **陆黎**：28 岁顶尖法医（物证/尸检/痕迹/逻辑），冷静锐利、不信命。证物库爆炸中为护证物被埋——冷开场 0.6–1.0s 出现，随即**同脸 match cut** 切古装。
- **苏昭华**：罪臣苏家嫡女，绝色、肤白。原主已在冲喜途中惊惧而亡，陆黎灵魂在洞房惊魂。
- **金手指=法医之眼，非全知**（★铁律）：能验伤/辨痕/读物证/以逻辑破局；**不懂**古代毒理成分、本朝秘辛、武功（前期），须靠查证与交易获得。观众看到 ≠ 昭华知道。
- **服装（★场景合理化露腿，用户 2026-07-18 锁）**：正式/朝堂/见长辈端庄；**只在合理场景露又白又细的腿**——沐浴/寝衣薄纱/涉水/受伤裹伤/习武短打/舞姬伪装/高开衩骑装。露腿服从剧情、写进 `{shot}`，不硬凹不出戏、SFW。每个女性角色都锁"端庄版 + 露腿版"两套定妆图。
- **人物弧**：弃子求生 → 王府立足 → 查苏家冤案 → 复仇卫崇 → 与裴珩先婚后爱 → 谋一个自己说了算的活法。

## 五、故事逻辑硬标准（★防崩人设/开挂/降智）
1. 法医之眼非全知；毒理成分/本朝秘辛/武功须靠查证交易获得，观众看到≠角色知道。
2. 反派为利益行动，不随身携带自证其罪之物、不降智送线索。
3. 爱情动作"一举多得"：亲密/救命须同时完成取证/遮挡/伪装/结盟/翻盘之一。
4. 裴珩装病有代价与破绽，不靠"其实一直无敌"洗白危机。
5. 翻案靠物证与互证，不靠反派自曝。
6. 背叛与反转须埋线、可回溯，不天降。

## 六、结构硬标准（每集必过，落地 /base/01）
- 单集 28–32s，目标 30s，30fps，每拍整数帧（成片基准 E001=12 拍/约 30s）。
- 0–1s 出钩子，命中 ≥2 触发器（钱与利害/好奇缺口/高唤醒/与我有关）。
- **前 3 秒必是动态视频**（kling I2V），完成不可逆动作 + 抛唯一主问题。
- 主问题答案不早于全片 80%；答案后再抛更危险新钩（勾下一集）。
- 每 3 秒至少一次变化；**动态视频拍 ≥3 段**、含开头钩子，**禁止连续 >2 拍纯图文**。
- 回环收尾接开头，诱二刷；每集末留追更悬念。

## 七、三行影视字幕（落地 /base/03）
每 utterance 三行同写：`pinyin`（字母声调）/`zh`（与中文配音逐字一致）/`local`（越南文意译）。
一屏一句短句，逐字/逐词卡拉OK。**出图/出视频 prompt 绝不写字幕/文字/边框/留白**（字幕由 Remotion 后期合成）。
