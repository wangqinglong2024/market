## 前三秒为什么能留人

1. 第 0 帧已经是红纱、交杯酒和现代 AI 林晚直视，不从空镜或片名开始。
2. 0.3 秒说出“仇人”，首秒同时出现婚姻与危险关系。
3. 1.4 秒毒已落入酒中，不是“准备下毒”。
4. 2.2 秒前后，萧玦明确看见下毒仍主动喝下，并反扣她手腕。
5. 观众三秒内只剩一个具体问题：他为什么故意喝毒？

## 24 秒权力变化

`她掌握毒酒 → 他看见仍喝并扣腕 → 她发现他早有准备 → 她拔簪复仇 → 他用半烧诏换合作 → 他用近身遮挡门缝 → 19.2秒后才演毒发 → 20.8秒说明反钓鱼 → 裴玄立即下灭口令`

爱情动作没有单独撒糖：拉近身体只为挡住门缝，低声靠近只为不让内应读到嘴型。
